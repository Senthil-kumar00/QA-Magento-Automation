- Login Test

- Change Password Test

- Selenium

## How to Runpip install -r requirements.txt
```bash
pip install -r requirements.txt
pytest tests/
```

## Folder Structure
- tests/: Test cases
- pages/: Page Object Model
- utils/: Config
- screenshots/: Store execution proofs
